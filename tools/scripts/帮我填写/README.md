# 📝 帮我填写 - 工作日报自动助手

> 自动化填写工作日报的智能工具，支持 AI 生成内容

## ✨ 功能特性

- 🤖 **AI 智能生成** - 基于 OpenAI GPT 自动生成工作日报内容
- 🌐 **自动网页填充** - 使用 Selenium 自动登录并填写日报系统
- 📝 **关键词驱动** - 只需输入关键词，AI 自动扩展成完整报告
- 📋 **TODO 管理** - 自动从关键词生成待办事项
- 💾 **历史记录** - 保存填写历史，支持查看和复用
- 🖥️ **图形界面** - 基于 Tkinter 的友好操作界面

## 🚀 使用方法

### 1. 安装依赖

```bash
pip install openai selenium
```

同时需要安装 Chrome 浏览器和 ChromeDriver。

### 2. 运行工具

```bash
python todo_gui.py
```

或双击运行 `install.bat`（Windows）

### 3. 配置

首次使用需要设置：
- OpenAI API Key
- 目标网站登录信息
- 其他个性化设置

配置保存在 `settings.json` 中。

### 4. 使用流程

1. 输入今日工作关键词（支持多行）
2. 点击"生成并填写"
3. AI 自动生成：
   - [TOP5] - 今日工作产出
   - [总结] - 工作概括
   - [TODO] - 待办事项
4. 自动填充到目标网站

## 📝 输入格式

关键词示例：
```
完成了游戏本地化审计工具的优化
修复了3个LQA相关的bug
参加了部门周会
整理了AI工具文档
```

## 📤 输出格式

自动生成的日报格式：
```
[TOP5]
- 完成游戏本地化审计工具的UI优化，提升了用户体验
- 修复3个关键bug，确保系统稳定运行
...

[总结]
今日主要完成工具优化和问题修复...

[TODO]
- 跟进用户反馈的问题修复
- 完成新版本功能测试
...
```

## ⚙️ 配置文件

`settings.json`：
```json
{
  "api_key": "sk-...",
  "prompt_template": "...",
  "target_url": "http://i.4399om.com/todo/list"
}
```

## 🔒 安全提醒

- API Key 仅保存在本地 `settings.json`
- 建议使用环境变量或密钥管理服务
- 不要将配置文件提交到公开仓库
